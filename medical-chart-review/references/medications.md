# Medication Review

## Medication reconciliation sources

Reconcile across:
1. EHR active med list
2. Provider note med list (may differ!)
3. Pharmacy fill history (Surescripts, PBM data)
4. Patient-reported (interview, BPMH)
5. Discharge med list (for transitions of care)
6. State PDMP (controlled substances)

Discrepancies = findings. Categorize as: omission, duplication, wrong dose/route/frequency, therapeutic duplication, drug-disease conflict, drug-drug interaction.

## High-risk medication classes (Beers Criteria 2023 — older adults)

- Anticholinergics (diphenhydramine, oxybutynin, TCAs)
- Benzodiazepines and Z-drugs in elderly
- First-gen antihistamines
- Skeletal muscle relaxants
- Long-acting sulfonylureas (glyburide)
- PPIs >8 weeks without indication
- NSAIDs in CKD/HF/anticoagulated patients
- Metoclopramide chronic use

## Always-check interactions / contraindications

- **Anticoagulants** (warfarin, DOACs) + NSAIDs / antiplatelets / certain antibiotics
- **Warfarin** + amiodarone, fluconazole, Bactrim, levothyroxine changes
- **SSRIs/SNRIs/MAOIs** + tramadol/triptans/linezolid → serotonin syndrome
- **Statins** + macrolides / fibrates / certain CCBs → rhabdo risk
- **QTc-prolonging** combos (ondansetron + methadone + macrolides + azoles)
- **K-sparing diuretics + ACEi/ARB + K supplements** → hyperkalemia
- **Metformin** + IV contrast in CKD; eGFR thresholds
- **Pregnancy Category D/X**: ACEi/ARB, statins, warfarin, isotretinoin, valproate, methotrexate
- **Renal dosing**: many abx, gabapentin, DOACs, allopurinol, metformin
- **Hepatic dosing**: acetaminophen ceiling 2g in liver disease

## Polypharmacy threshold

- ≥5 chronic meds = polypharmacy (some define ≥10 = hyperpolypharmacy)
- Flag for deprescribing review

## Controlled substances red flags

- Multiple prescribers / multiple pharmacies (doctor/pharmacy shopping)
- Early refills
- Opioid + benzodiazepine combo (FDA black box)
- MME/day >50 (caution) or >90 (high-risk per CDC 2022 guideline)
- No PDMP check documented
- No naloxone co-prescribed at high MME

## Allergy review

- Distinguish true allergy (anaphylaxis, SJS, hives) from intolerance (nausea, headache)
- "PCN allergy" — most are non-IgE; cross-reactivity with cephalosporins is much lower than historically taught (~1–2%)
- Sulfa: antibiotic sulfas ≠ non-antibiotic sulfas (e.g., furosemide, celecoxib)
